# Многоликая вера России

Учебный сайт-проект о религиозном многообразии России — православие, ислам,
буддизм и иудаизм: история появления, повседневный уклад, межрелигиозный диалог
и интерактивная конфессиональная карта.

## Стек
Чистый HTML/CSS/JS без сборки. Анимации — Lenis + GSAP/ScrollTrigger,
карта — Leaflet (всё через CDN).

## Локально
Открыть `index.html` в браузере или поднять статический сервер:

```bash
python3 -m http.server 8000
```
